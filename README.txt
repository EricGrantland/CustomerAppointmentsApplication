Running the program:
-login using the following credentials: username: test password: test
-In the main menu, you will see any appointments scheduled on the current day. You have the following three options:
    -"Manage Appointments"- allows you to add a new appointment, update a current appointment, or delete an appointment.
    -"Manage Customers"- allows you to add a new customer, update a current customer's info, or delete a customer
    -"Reports"- pulls up a page with three different reports, including:
        -The number of appointments sorted by Month and Type
        -The appointment schedule for each contact in the database (select which contact you would like to view)
        -*All scheduled appointments for a given customer (select which customer you would like to view)
-You may exit the program by using the "Exit" button, if shown on the bottom right-hand corner of the window.

*- indicates extra report
